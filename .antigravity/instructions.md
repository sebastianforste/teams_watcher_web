# Teams Watcher Web Project Instructions

## Project Context
A Next.js web interface for monitoring and controlling the Teams Watcher script.

## Technology Stack
- **Frontend**: Next.js with TypeScript
- **Styling**: Modern CSS (glassmorphism, dark mode)
- **Backend**: API routes for controlling the watcher

## Architecture
- `app/`: Next.js app router pages
- `components/`: Reusable UI components
- `api/`: Server-side API endpoints

## Key Considerations
- SSR for performance
- Real-time status updates
- Responsive design for mobile
